# Catálogo de Libros JavaFX

Aplicación de escritorio desarrollada en Java y JavaFX para gestionar
un catálogo de libros de una biblioteca escolar.

## Funcionalidades

* Registro de libros
* Edición de libros
* Eliminación de libros
* Visualización en tabla
* Pantalla de detalle
* Exportación de reporte a CSV

## Persistencia

Los datos se guardan en un archivo CSV local ubicado en `data/catalogo.csv`.

## Tecnologías

* Java
* JavaFX
* FXML
* CSV para almacenamiento
